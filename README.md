## Run Locally

Open the folder in IntelliJ and open `index.html` in a browser.

## GitHub Pages Deployment

1. Push all files to a GitHub repository.
2. Go to Settings → Pages.
3. Choose Deploy from branch.
4. Select `main` and `/root`.
5. Save.

## Netlify Deployment

1. Create a new Netlify site.
2. Connect the GitHub repository or drag in the project folder.
3. Leave build command blank.
4. Set publish directory to `/`.
5. Deploy.
